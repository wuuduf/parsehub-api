# ParseHub 调试版快捷指令

文件：`ParseHub-Debug.shortcut`

调试版会在以下节点停下来显示数据：

1. 分享输入或剪贴板内容。
2. 请求接口地址。
3. HTTP 原始响应和响应类型。
4. `data` 字段原始值和类型。
5. 文案、服务端媒体计数、快捷指令数组计数。
6. 完整图片 URL 数组和视频 URL 数组。
7. 文案复制完成。
8. 每张图片下载前、下载后、快速预览、写入图库。
9. 每个视频下载前、下载后、快速预览、写入图库。

遇到异常时，请记录最后一个正常节点，并截图该节点与下一个错误弹窗。调试截图中的媒体签名 URL 有有效期，不建议公开长期分享。
